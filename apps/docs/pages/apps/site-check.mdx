# Site-Check SEO

Outil d'analyse SEO gratuit pour auditer les sites web.

## Routes

| Route | Description |
|-------|-------------|
| `/` | Page d'entrée avec formulaire |
| `/[url]` | Résultats d'analyse |

## Catégories d'analyse

### Performance
- Temps de chargement
- Core Web Vitals
- Taille des ressources

### SEO
- Balises meta
- Structure des titres
- Données structurées
- Open Graph

### Sécurité
- HTTPS
- En-têtes de sécurité
- Cookies

### Accessibilité
- Attributs ARIA
- Contraste des couleurs
- Navigation clavier

### Mobile
- Responsive design
- Viewport
- Éléments tactiles

## Composants

- `AnalysisCard` - Carte de résultat
- `ScoreGauge` - Jauge de score
- `OverallScore` - Score global
- `OpenGraphPreview` - Prévisualisation OG
- `MetaTagsViewer` - Visualisation meta
- `JsonLdViewer` - Visualisation JSON-LD
